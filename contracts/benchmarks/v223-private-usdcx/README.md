# V2.23 Private USDCx Benchmarks

This directory stores Phase 1 benchmark artifacts for the `v2.23` private `USDCx` feasibility spike.

Expected files:

- `spike-*.json`
- `benchmark-suite-*.json`
- optional `summary.md`

Each artifact should make it easy to answer:

- which scenario was tested
- whether bid amount leakage was observed
- how many records and credentials were required
- what proving time, wall-clock time, and fee cost were observed
- what trust assumptions remained in the tested custody model

Primary references:

- `docs/contracts/v2.23-phase-1-feasibility-spike.md`
- `docs/contracts/v2.23-custody-decision.md`
- `docs/contracts/v2.23-benchmark-findings.md`
