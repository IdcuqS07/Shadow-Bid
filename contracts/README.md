# ShadowBid Contracts

This directory contains the active ShadowBid `v2.20` Leo program.

## Contents

- `src/main.leo` - marketplace logic for commit-reveal auctions, reserve handling, settlement, and disputes
- `program.json` - Leo program metadata

## Build

```bash
cd contracts
leo build
```
